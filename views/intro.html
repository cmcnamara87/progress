Setup

Step 1 - Download and Install

  1. Download Mac App
  2. Drag the app to the Applications folder.
  3. Run the app and sign in.

Step 2 - Add a Project

Add a project from the menubar app.

Step 3 - Start working

Whenever you save a file in one of your projects folders, Progress will start tracking. The menubar icon will change to a box with an arrow pointing into it.

Step 4 - Upload a Picture to your Diary

Whenever you are working, use CMD + SHIFT + 4 to take and upload a screenshot.
